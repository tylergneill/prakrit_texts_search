# Search for Prakrit Digital Text Project 

Forked from https://github.com/aso2101/prakrit_texts.

First convert TEI XML (`tei` folder) to HTML (`html` folder) using a script (`scripts/tei-to-html.py`).

Then point Pagefind to HTML (output `html/pagefind`).

Finally, add some transliteration and pagination sugar.

In all respects following pattern of https://github.com/tylergneill/ngmcp-wiki-mirror.

Co-created with Gemini CLI.
