# Search interface for Prakrit Digital Text Project 

Forked from https://github.com/aso2101/prakrit_texts.

Served using GitHub Pages at https://tylergneill.github.io/prakrit_texts_search.

# How

First converted TEI XML (`tei` folder) to HTML (`html` folder) using a script (`scripts/tei-to-html.py`).

Then pointed Pagefind to HTML (output `docs/pagefind`).

Finally, add some transliteration and pagination sugar,
plus a little diagram of the relevant file structure.

In all respects following pattern of https://github.com/tylergneill/ngmcp-wiki-mirror.

Co-created with Gemini CLI.
